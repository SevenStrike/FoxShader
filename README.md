# FoxShader 高级着色器包
这是一套基于 Unity ShaderGraph 的高可定制、模块化材质特效资源包。所有功能均拆分为独立的 SubGraph 与 Master Graph，开发者可以按需拖入项目、自由拼装，快速实现从车漆、玻璃、建筑内饰到科幻扫描、流动噪波等 30 + 常见视觉效果
